INSTALACIÓN Y DEPLOYMENT
================================================================================

OPCIÓN 1: Instalador Profesional (Recomendado)
  Archivo: FincaFacil_Installer_v2.0.exe
  Tamaño: ~250 MB
  Proceso: Setup Wizard automático
  Resultado: Instalación en C:\Program Files\FincaFácil
  Accesos: Escritorio + Menú Inicio

OPCIÓN 2: Ejecutable Standalone
  Archivo: FincaFacil.exe
  Tamaño: ~200-300 MB
  Proceso: Ejecutar directamente
  Resultado: No requiere instalación
  Ventaja: Portátil (USB, etc)

OPCIÓN 3: Desde Código Fuente
  Requisitos: Python 3.11+
  Instalación: pip install -r src/utils/requirements.txt
  Ejecución: python src/main.py
  Compilación: python scripts/build_exe_simple.py

FLUJO DE USO
================================================================================

1. INSTALACIÓN
   → Ejecutar FincaFacil_Installer_v2.0.exe
   → Seguir wizard
   → Crear accesos directos
   
2. PRIMER INICIO
   → Pantalla de login aparece
   → Usuario registra cuenta (usuario + contraseña)
   → Licencia de prueba se activa automáticamente (6 meses)
   
3. TOUR INTERACTIVO
   → 8 pasos guiados opcional
   → Puede omitirse en cualquier momento
   → Disponible nuevamente desde Ajustes
   
4. USANDO LA APLICACIÓN
   → Crear fincas y animales
   → Registrar eventos de salud, reproducción, leche
   → Generar reportes
   → Hacer copias de seguridad
   
5. DESPUÉS DE 6 MESES
   → Solicitar código de activación
   → Ingresar código en Ajustes → Estado de Licencia
   → Licencia se actualiza a permanente
   → Uso sin límite de tiempo
