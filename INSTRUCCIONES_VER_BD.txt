OPCIÓN 1: DB Browser for SQLite (RECOMENDADO)
----------------------------------------------
Es la herramienta más fácil y completa para ver bases de datos SQLite.

1. Descarga e instala DB Browser for SQLite:
   https://sqlitebrowser.org/
   (Es gratuito y de código abierto)

2. Abre DB Browser for SQLite

3. Haz clic en "Open Database" (Abrir Base de Datos)

4. Navega a: C:\Users\lenovo\Desktop\FincaFacil\database\fincafacil.db

5. ¡Listo! Podrás ver todas las tablas, datos, y ejecutar consultas SQL


OPCIÓN 2: Script Python Simple
-------------------------------
He creado un script para ver la BD desde la terminal:

1. Ejecuta: python ver_base_datos.py

2. Te mostrará todas las tablas y sus datos


OPCIÓN 3: Extensión de VS Code
--------------------------------
Si usas Visual Studio Code:

1. Instala la extensión "SQLite Viewer" o "SQLite"

2. Abre el archivo: database/fincafacil.db

3. Podrás ver la estructura y datos directamente en VS Code


OPCIÓN 4: Desde Python Interactivo
----------------------------------
Abre Python y ejecuta:

    import sqlite3
    conn = sqlite3.connect('database/fincafacil.db')
    cursor = conn.cursor()
    
    # Ver tablas
    cursor.execute("SELECT name FROM sqlite_master WHERE type='table'")
    print(cursor.fetchall())
    
    # Ver datos de una tabla
    cursor.execute("SELECT * FROM animales LIMIT 5")
    print(cursor.fetchall())
    
    conn.close()
