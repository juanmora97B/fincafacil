CÓMO USAR FINCAFÁCIL
================================================================================

INSTALACIÓN:
  1. Ejecutar FincaFacil_Installer_v2.0.exe
  2. Seguir el Setup Wizard
  3. Crear accesos directos
  4. Listo para usar

PRIMER USO:
  1. Ejecutar FincaFácil desde Menú Inicio
  2. Crear cuenta (usuario + contraseña)
  3. Licencia de prueba se activa automáticamente
  4. Tour interactivo (opcional)
  5. Comenzar a usar

ACTIVACIÓN:
  1. Después de 6 meses, solicitar código
  2. Ir a Ajustes → Estado de Licencia
  3. Ingresar código de activación
  4. Licencia actualizada a permanente
